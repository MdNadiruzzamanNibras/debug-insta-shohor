Local Storage vs Session Storage Difference

The main difference in this storage is that when tab or browser is closed local storage data is stored but session storage data is clear. Local storage doesn't work specific tab but session storage does it. Local storage data range is high but session storage data range is low. Local storage capacity is 10 mb and session storage capacity is 5 mb. If you need to clear local storage data you have to delete it but session storage is not need. The browser usually stores the data that is needed later it store local storage and the browser stores the data that is not needed later it store session storage.
